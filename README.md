# discuz-api

使用 Koa2 构建一个 web 网站，探讨相关的技术。

## 配置

将`config/constants.dist.js`修改为`config/constants.js`, 在其中配置需要的相关真实环境。

```
npm install
npm run start
```

打开`http://localhost:3000/test`
